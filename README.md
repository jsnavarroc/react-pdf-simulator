

# Pdf Simulator
## Como instalar la libreria. 

Se instala así: 
    
    yarn add react-pdf-simulator

Se carga así: 

    import React from 'react';
    import PDFsimulator from 'react-pdf-simulator';

    const index = () => {
        return (
            <PDFsimulator>
                <h1>¡¡Hola Mundo!!</h1>
                <h2>Coloca todo tu codigo HTML o JSX aquí</h2>  
            </PDFsimulator>
        );
    };

    export default index;
